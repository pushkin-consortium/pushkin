This is the documentation for the Pushkin package. To view the documentation, visit [this link](https://languagelearninglab.gitbook.io/pushkin/).

### Using and editing the new documentation 

1. Install mkdocs-material in a virtual environment: `pip install mkdocs-material`
2. Install mike in this same environment: `pip install mike`
3. Deploy documentation locally and view it: `mike serve`
